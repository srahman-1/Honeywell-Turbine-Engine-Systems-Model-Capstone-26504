# Turbofan Engine Modeling Capstone

This repository contains software and modeling tools used for the
UofA Capstone Turbofan Engine Systems Modeling & Simulation project.

## Contents
scripts/  -> Python modeling scripts
matlab/   -> MATLAB cycle analysis tools
data/     -> CSV results from trade studies
results/  -> plots and figures
docs/     -> project documentation

## Main Model
Advanced_Turbofan_Cycle_TradeStudy.py performs parametric cycle
analysis for turbofan engines including BPR and OPR trade studies.
